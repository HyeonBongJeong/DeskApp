## :mag: Project

#### 개요
사내에서 사용하는 업무메신저를 확장하여 각각의 프로젝트에 인원을 추가할수 있고 진행상황과 서로의 스케줄을 공유하기 위해 만들었습니다. 

#### 주요기능
- 실시간 접속여부확인
- 1대1채팅과 알림 전송
- 프로젝트 추가, 수정, 열람 권한부여
- 로그인 아이디는 일정한 규칙으로 자동 부여
